HowTo use Homebrew
by Rob Williams

REF: https://brew.sh/

# Install
`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"`

# Update
`brew update`

# Upgrade installed packages
`brew upgrade`

# List installed packages
`brew list`
